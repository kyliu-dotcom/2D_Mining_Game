Project Title: Labyrinth
Description: 


This game follows an adventurer who journeys into an ever changing maze in search of treasure. The adventurer is represented by a sprite, which has the ability to mine blocks and open doors (given that it has a key). By mining blocks, the adventurer collects currency, which can be redeemed at the store to buy a key that allows the player to proceed to the next area. A player’s progress is marked by how close it is to the treasure chest, which is indicated by the types of keys the player has.
The ultimate goal of the player is to get to the end of the map and reach the treasure, which will give the player the time they took to get to the treasure. This time can be used as a progress check for the future times they play the game so that they can see how their speed and reflexes improve.

The two most important installations needed in this project are cmu_graphics and PIL. To install cmu_graphics, go to this link, https://academy.cs.cmu.edu/desktop, and click download. Once the file is downloaded, move it to the submission folder of my project (Final Draft) and store that in the src folder within the Final Draft folder in order to run cmu_graphics. Make sure it is in that exact folder, or the code won't be able to find it
For installing PIL, open the command terminal on the IDE that is running the code, then issue this command: pip install Pillow. If loaded successfully, then the PIL library should be good to use in the code. 
Shortcut: The only current shortcut in the code right now is the use of the ‘r’ button, which quickly resets all conditions so that the player can start over.
